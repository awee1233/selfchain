# This is instructions for automation installation 


- Create Folder
``` 
mkdir self
cd self
``` 
- Clone repositorie
``` 
git clone https://github.com/awee1233/selfchain/
```
- Make it executable
```
cd selfchaind
```
```
chmod +x autoinstall.sh
```

- AND RUN THE SCRIPT
```
./autoinstall.sh
```
